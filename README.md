# rubick
## Running

- `git clone git://github.com/abia321/rubick.git`
- `cd rubick`
- `mvn install`
- `cd target`
- `java -jar rubick-1.0.jar --cluster.name="elasticsearch" --cluster.ip="127.0.0.1" --cluster.port=9200 --cluster.transport=9300   ` 


- `open` <http://localhost:8080/base/index>
